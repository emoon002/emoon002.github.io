Photos From Hubble

This interactive photo gallery uses CSS3 and jQuery to create fluid image effects using the transform and transition properties. I also used CSS3's multiple column properties to create a two column layout for the gallery text.

  • Relative Units

  • Borders

  • Box-Shadows

  • Ordered Lists
