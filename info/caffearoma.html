Caffè Aròma Website

I created this website for a local business in Newbury Park, CA called Caffè Aròma. This is a static website built using HTML5, CSS3 and JavaScript. It features a home page, menu page, location page, and about page. Highlights include a custom designed viewable/downloadable PDF menu, Google Maps location integration, custom graphics, and a rotating WS3-CSS image carousel on the About page.

Website Features:

  • Responsive Design
  • Custom PDF Menu
  • Custom Graphics
  • Google Maps Integration
  • Rotating Image Carousel
