Interactive Login Form

I created this interactive login form to learn more about jQuery's power and functionality when building dynamic web applications.
I experimented with commonly used jQuery methods and event handlers such as .hide(), .show(), .prop(), .val(), .focus(), .keyup(), and .next(). This project was a good review of fundamental programming concepts like writing modular code, as well as basic DOM traversal using jQuery methods.

Website Features:

  • HTML Input Form
  • Dynamic Pop-up Hints
  • String Comparison
  • String Length Validation
  • Event Handling

Languages and Tools:

  • HTML5
  • CSS3
  • jQuery
