Las Vegas Travel Website

I built this responsive Las Vegas themed travel website to learn more about advanced CSS techniques such as text-shadows, color gradients, borders, box-shadows, and more. Additionally, I wanted to experiment further with responsive web design using media queries to optimize websites for display on mobile devices and varying screen sizes.

Website Features:

CSS techniques used:

  • Box-sizing
  • Color Gradients
  • Relative Units
  • Text-Shadows
  • Borders
  • Media Queries
  • Box-Shadows
  • Floats
  • Lists

Languages and Tools:

  • HTML5
  • CSS3
  • Photoshop
